**Инструкция:** Скопируйте весь текст ниже и вставьте его в чат Cursor (Ctrl+L / Cmd+L) или в Composer (Ctrl+I / Cmd+I).

### ROLE:
Ты — Senior Full-stack Developer и UI/UX Designer, специализирующийся на современных AI-интерфейсах. Твоя задача — создать высокотехнологичный одностраничный сайт-портфолио для AI-инженера Татьяны.

### TECH STACK:
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Icons: Lucide React
- Animations: Framer Motion
- UI Components: Shadcn UI (по возможности)

### DESIGN CONCEPT: "Cyber-Business Minimal"
Стиль должен сочетать профессионализм бизнес-решений и футуристичную эстетику AI.
- **Color Palette**: 
  - Background: `#0A0A0B` (Deep Dark)
  - Primary Accent: `#00F2FF` (Neon Cyan)
  - Secondary Accent: `#7000FF` (Electric Purple)
  - Text: `#F8FAFC` (Off-white)
- **Typography**: 
  - Headings: 'Space Grotesk' или 'Inter' (Bold/ExtraBold)
  - Body: 'Inter' (Regular/Medium)
- **Visual Elements**: 
  - Bento-grid layout для кейсов.
  - Glassmorphism эффекты (backdrop-blur) для карточек и навигации.
  - Тонкие анимированные границы (border-gradient).
  - Мягкое свечение (glow) вокруг ключевых элементов.

### STRUCTURE & CONTENT:

#### 1. Hero Section (Шапка)
- **Visual**: Слева — текст, справа — место под аватарку (используй стильный плейсхолдер или рамку в стиле киберпанк).
- **Title**: "Татьяна: Ваш AI-специалист для роста бизнеса"
- **Subtitle**: "Помогаю предпринимателям и компаниям автоматизировать рутину, экономить время и увеличивать прибыль с помощью умных нейросетей. Без сложных терминов, с фокусом на результат."
- **CTA Button**: "Обсудить проект" (плавный градиент при наведении).

#### 2. Value Proposition (Польза)
- Сетка из 4-5 карточек с иконками (Lucide):
  - Экономия времени (до 20 часов в неделю).
  - Сокращение расходов.
  - Увеличение прибыли.
  - Конкурентное преимущество.
  - Оптимизация процессов.

#### 3. Portfolio Cases (Кейсы) — Bento Grid Layout
Оформи кейсы в виде стильной сетки разного размера. Для каждого кейса предусмотри место под изображение (баннер).
- **Case 1: AI-Генератор постов для VK**
  - Задача: Ручное создание контента, падение охватов.
  - Результат: Охваты +30%, заявки +15%, время -80%.
- **Case 2: RAG-система (База знаний)**
  - Задача: Медленный поиск информации в документах.
  - Результат: Скорость поиска +40%, точность ответов 90%.
- **Case 3: AI-мониторинг конкурентов (CompetitorAI)**
  - Задача: Трудоемкий ручной анализ рынка.
  - Результат: Автоматический сбор данных, быстрые стратегические решения.

#### 4. Footer & Contact
- Оставь только один канал связи: Telegram (@ваш_телеграм).
- Призыв: "Напишите мне прямо сейчас, чтобы обсудить ваш проект и получить бесплатную консультацию!"

### ANIMATION REQUIREMENTS:
- Плавное появление (fade-in-up) секций при скролле.
- Hover-эффект "увеличения" и "свечения" для карточек кейсов.
- Интерактивный фон (тонкие линии или частицы, не отвлекающие от текста).
